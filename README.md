# 🛡️ BotGuard SOC Monitoring Lab

🚀 Personal Cybersecurity Home Lab – SOC Monitoring & Threat Detection

The BotGuard SOC Lab is a Raspberry Pi–based monitoring environment designed to simulate a real-world Security Operations Center (SOC).

The lab collects system metrics, network telemetry, and security logs, then visualizes them through centralized dashboards to detect suspicious activity and potential attacks.

This project demonstrates practical cybersecurity engineering skills including monitoring infrastructure, log aggregation, and threat detection.

---

# 🧱 Architecture Overview

SOC Monitoring Stack

Raspberry Pi 5 (SOC Control Node)
│
├── Prometheus – metrics collection
├── Grafana – visualization dashboards
├── Loki – centralized log aggregation
├── Zeek – network monitoring
│
└── Raspberry Pi 4 (Worker Node)
    └── Node Exporter – system telemetry

The control node collects and visualizes telemetry from worker nodes across the network.

# 📊 SOC Monitoring Dashboard

The Grafana dashboards provide real-time visibility into system performance, network activity, and potential security threats.

Monitored metrics include:

System telemetry  
• CPU usage  
• RAM usage  
• Disk utilization  
• Disk I/O  
• CPU temperature  
• system uptime  

Network telemetry  
• inbound / outbound traffic  
• packet rates  
• active TCP connections  
• network spikes  

Security telemetry  
• SSH login monitoring  
• brute force detection  
• port scan detection  
• suspicious IP tracking  

---

# 🔐 Security Detection Panels

Security dashboards include monitoring panels designed to detect suspicious behavior.

Examples include:

• SSH Brute Force Detector  
• Top SSH Source IPs  
• SSH Port Scan Activity  
• Suspicious SSH Source IP Tracking  
• Threat Activity Heatmap  

These panels simulate how a SOC identifies suspicious login attempts and intrusion activity.
